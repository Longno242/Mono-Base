# Security Policy

## Supported versions

Only the latest commit on `master` is supported. This is a hobby / modding base project with no formal release cycle.

## Reporting a vulnerability

If you find a security issue in **this repository's code** (not in a game you injected into):

1. **Do not** open a public GitHub issue for exploitable bugs
2. Use [GitHub Security Advisories](https://github.com/Longno242/Mono-Base/security/advisories/new) or contact the repo owner privately

Include:

- What the issue is
- Steps to reproduce (if applicable)
- What you think the impact is

I'll respond when I can and work on a fix.

## Out of scope

This project is a local DLL injection base for single-player Mono Unity games. Reports about:

- Cheating in multiplayer games
- Bypassing anti-cheat in commercial titles
- Abusing injected code against other players

...will not be handled here.

## Disclaimer

You use this software at your own risk. The author is not responsible for bans, crashes, or data loss from injecting into games.
