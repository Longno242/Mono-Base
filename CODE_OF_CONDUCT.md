# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our standards

Examples of behavior that contributes to a positive environment:

- Being respectful in issues, pull requests, and discussions
- Giving constructive feedback
- Accepting that people use this project in different ways and on different games

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Publishing other people's private information
- Encouraging cheating in multiplayer or online games
- Trolling or deliberately disruptive comments

## Scope

This applies in project spaces (GitHub issues, PRs, discussions) and when someone is representing the project publicly.

## Enforcement

Report issues to the repository owner through GitHub. All reports are reviewed and handled as needed.

This code of conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
